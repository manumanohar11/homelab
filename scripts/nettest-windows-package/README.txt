================================================================================
                    NETWORK TEST TOOL - FOR WINDOWS
================================================================================

This tool tests your internet connection and creates a report you can share.

--------------------------------------------------------------------------------
                            QUICK START
--------------------------------------------------------------------------------

   1. Install Docker Desktop (only needed once):

      - Go to: https://www.docker.com/products/docker-desktop
      - Click "Download for Windows"
      - Run the installer, click Next/Yes on everything
      - RESTART YOUR COMPUTER when asked
      - After restart, Docker will start automatically (wait 1-2 minutes)

   2. Double-click: CLICK_ME_TO_TEST.bat

   3. Wait for the test to finish (2-5 minutes first time)

   4. Your report opens automatically in your browser!

   5. Share the HTML file from the "output" folder


--------------------------------------------------------------------------------
                            TROUBLESHOOTING
--------------------------------------------------------------------------------

"Docker is not running"
   - Make sure Docker Desktop is installed
   - Look for the whale icon in your system tray (bottom right)
   - If not there, search "Docker Desktop" in Start menu and open it
   - Wait 1-2 minutes for it to start, then try again

"Windows protected your PC" warning
   - Click "More info"
   - Click "Run anyway"
   - This is normal for downloaded scripts

First run is slow
   - First time takes 2-5 minutes to download components
   - After that, tests take about 1 minute


--------------------------------------------------------------------------------
                         WHAT GETS TESTED
--------------------------------------------------------------------------------

   - Download/Upload speed
   - Ping latency to Google, Cloudflare, Microsoft
   - Connection stability (jitter, packet loss)
   - Bufferbloat (lag under load)
   - VoIP call quality score


--------------------------------------------------------------------------------
                         SHARING RESULTS
--------------------------------------------------------------------------------

After the test, find your report in the "output" folder:

   output\nettest_report_YYYYMMDD_HHMMSS.html

Send this file via WhatsApp, email, or any way you like.
Your friend can open it in any browser - no installation needed to VIEW reports.
